# airline_passenger_satisfaction.xlsx
This is an Excel project depicting replication of the whole data table using Vlookup; using Pivot to show satisfaction level of travellers based on gender & class & also a slicer showing customer type; visualization using a combo graph for passenger satisfaction for an airline.
Data sourse: https://mavenanalytics.io/data-playground/airline-passenger-satisfaction
